# Data Fusion Contest 2026 — Задача 2 «Киберполка»

Multi-label классификация 41 финансового продукта клиентов банка.

## Постановка задачи

Клиенты банка владеют разными банковскими продуктами — различными видами счетов, карт и услуг. Необходимо предсказать вероятность открытия каждого из 41 продукта для каждого клиента.

В отличие от классических задач рекомендаций, бизнес интересуют именно вероятности открытия каждого продукта, а не ранжирование.

## Данные

1,000,000 клиентов с полностью анонимными и обфусцированными признаками:
- **750,000** — тренировочные (с разметкой)
- **250,000** — тестовые

### Файлы

```
data/
├── train_main_features.parquet    # 750k × 200 (67 cat + 132 num + customer_id)
├── train_extra_features.parquet   # 750k × 2242 (2241 num + customer_id)
├── train_target.parquet           # 750k × 42 (41 target + customer_id)
├── test_main_features.parquet     # 250k × 200
├── test_extra_features.parquet    # 250k × 2242
├── sample_submit.parquet          # пример посылки
└── baseline_catboost.ipynb        # ноутбук базового решения
```

### Признаки

- `cat_feature_1` ... `cat_feature_67` — категориальные (int32)
- `num_feature_1` ... `num_feature_132` — числовые (float64) — основной набор
- `num_feature_133` ... `num_feature_2373` — числовые (float64) — дополнительный набор
- Много пропущенных значений и выбросов
- Описания признаков не предоставляются

### Целевые переменные

41 бинарный таргет, сгруппированный по продуктам:

| Группа | Таргеты |
|--------|---------|
| 1 | target_1_1 ... target_1_5 |
| 2 | target_2_1 ... target_2_8 |
| 3 | target_3_1 ... target_3_5 |
| 4 | target_4_1 |
| 5 | target_5_1, target_5_2 |
| 6 | target_6_1 ... target_6_5 |
| 7 | target_7_1 ... target_7_3 |
| 8 | target_8_1 ... target_8_3 |
| 9 | target_9_1 ... target_9_8 |
| 10 | target_10_1 |

- Сильный дисбаланс классов
- Заметная корреляция внутри групп, target_10_1 отрицательно коррелирует с остальными

## Метрика

**Macro Averaged ROC-AUC** — среднее ROC-AUC по каждому из 41 класса:

```python
from sklearn.metrics import roc_auc_score
roc_auc_score(y_true, y_pred, average="macro")
```

## Формат решения

Parquet-файл с 42 столбцами: `customer_id` + 41 предсказание (`predict_i_j` для каждого `target_i_j`).
Предсказания — raw scores (не вероятности), для всех 250,000 тестовых клиентов.

```
customer_id, predict_1_1, predict_1_2, ..., predict_10_1
1750001,     -4.559634,   -5.490853,   ..., -0.624437
```

## Public / Private split

- **Public** (30%): 75,000 клиентов
- **Private** (70%): 175,000 клиентов

Победители определяются по private. Можно выбрать до 2 финальных решений.
