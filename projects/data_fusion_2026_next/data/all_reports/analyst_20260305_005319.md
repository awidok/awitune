### 4. Data Quality Issues

**Issue**: Some correlation calculations showed values outside [-1, 1] when using numpy's masked arrays.

**Root Cause**: Numerical precision issues with features having very high NaN rates (99%+) or very few valid observations.

**Solution**:
```python
# Use robust correlation calculation
def robust_correlation(x, y, min_samples=100):
    valid_mask = ~np.isnan(x) & ~np.isnan(y)
    if np.sum(valid_mask) < min_samples:
        return np.nan  # Not enough data for reliable correlation

    x_valid = x[valid_mask]
    y_valid = y[valid_mask]

    # Use Spearman for robustness
    from scipy.stats import spearmanr
    corr, pvalue = spearmanr(x_valid, y_valid)
    return corr
```
